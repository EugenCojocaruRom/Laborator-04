Lab04 - Exercise 2 - Stack class

Define a Stack class for stacks of type int, represented by intrinsic vectors.

Create the following methods:

&nbsp;- constructor without parameters (default stack size 100)

&nbsp;- constructor with 1 parameter (stack size)

&nbsp;- void Push(int) - add a given int to the stack

&nbsp;- int Pop() - remove the element at the top of the stack

&nbsp;- boolean isEmpty() - check that the stack is empty

Check the operations in a static main method, included in the Stack class.
