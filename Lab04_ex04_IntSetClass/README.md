Lab04 - Exercise 4 - IntSet class

Define an IntSet class for numbers of type int, containing an intrinsic array of integers.



Create the following methods:

&nbsp;- constructor with 1 parameter (array size)

&nbsp;- void Add(int) - add an element to the array, if it doesn't exist

&nbsp;- boolean Contains() - check that a given number is in the array

&nbsp;- String toString() - the array converted into a string of characters

Create an array using the IntSet class and test the operations above by successive additions and displays.
